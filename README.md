# RugMania

A fast-paced on-chain survival game where you climb through levels by choosing the right door. Simple rules, smooth UI, and a fun risk-based progression loop.

## Features

- Clean, responsive UI
- Level-based progression
- Wallet login or demo mode
- Fully client-side game logic (contracts coming later)
- Built with modern Web3 tooling

## Tech Stack

- Next.js + TypeScript  
- Tailwind CSS  
- Framer Motion  
- Wagmi + Viem  
- Privy Auth  
- Mantle Testnet (future integration)

npm install
npm run dev
